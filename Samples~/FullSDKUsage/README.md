# Full SDK Usage

전체 기능 흐름을 빠르게 확인하는 샘플입니다.

- 게스트 로그인
- 이벤트 전송
- RemoteConfig 새로고침/폴링
- Edge Function 호출

## 실행 전 체크

1. `TrueSoft > Supabase > Create Settings Asset`
2. `projectUrl`, `publishableKey` 입력
3. `Assets/Resources/SupabaseSettings.asset` 로 저장
4. 필요하면 `TrueSoft > Supabase > Create Runtime Object In Scene`
