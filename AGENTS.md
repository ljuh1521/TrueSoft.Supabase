# 에이전트·자동 생성 공통 지침

## Git 커밋 메시지 언어 (최우선)

- **커밋 제목과 본문은 기본적으로 한국어로만 작성한다.** (소스 제어 패널의 커밋 메시지 자동 생성 포함.)
- **영어 한 줄 요약** (`Enhance documentation for...`, `Added detailed XML comments...` 등)을 **기본으로 쓰지 않는다.**
- 그런 문장이 나오면 **한국어로 의역**한다. 저장소 루트 `.cursorrules`에 나쁜 예·좋은 예가 있음.
- 사용자가 **영어로 작성해 달라고 명시한 경우에만** 영어를 사용한다.
